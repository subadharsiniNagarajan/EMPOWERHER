# 🚀 EmpowerHer - Quick Start Guide

## Instant Setup (30 seconds)

1. **Open the app**
   ```
   Double-click index.html
   ```

2. **Select language**
   - Choose from 6 languages
   - Click your preferred option

3. **Enter your name**
   - Type your name
   - Click "Start My Journey"

4. **Start exploring!**
   - You're now on the dashboard
   - Click any feature card to begin

## 🆘 Troubleshooting

**Language not switching?**
- Refresh the page
- Clear browser cache

**Voice input not working?**
- Check microphone permissions
- Try Chrome or Edge browser
- Ensure internet connection

**Mobile layout issues?**
- Rotate to portrait mode
- Zoom out if needed
- Refresh the page
